# Общая информация о приложении

**Happ** — кроссплатформенное программное решение для создания, настройки и управления пользовательскими профилями подключения, работающими на основе технологии **Xray**.

Приложение предназначено для мобильных устройств и настольных систем и предоставляет локальные инструменты для работы с параметрами соединения. Пользователь может самостоятельно формировать профили подключения, импортировать подготовленные конфигурационные файлы, изменять отдельные параметры, сохранять различные варианты настроек и выбирать необходимую конфигурацию в зависимости от поставленной задачи.

В основе архитектуры Happ находится система отдельных конфигурационных профилей. Каждый профиль представляет собой самостоятельный набор данных, включающий параметры соединения, правила маршрутизации, настройки обработки сетевых потоков и дополнительные управляющие элементы.

Такой принцип организации позволяет одновременно хранить несколько независимых конфигураций, редактировать их без влияния друг на друга и быстро переключаться между сохранёнными вариантами без необходимости повторной настройки.

---

# Управление конфигурационными профилями

Happ предоставляет инструменты для полноценной работы с пользовательскими настройками:

* создание новых профилей подключения;
* загрузка конфигураций из локальных файлов;
* редактирование сохранённых параметров;
* хранение нескольких независимых наборов настроек;
* настройка правил маршрутизации;
* изменение дополнительных параметров соединения;
* выбор активного профиля;
* обновление существующих конфигураций;
* управление всеми операциями непосредственно на устройстве.

Каждый профиль сохраняется отдельно от остальных, что позволяет создавать различные варианты параметров для разных сценариев использования и изменять выбранную конфигурацию без затрагивания других сохранённых данных.

# Локальное хранение данных

Все пользовательские профили и импортированные конфигурационные материалы сохраняются только во внутреннем хранилище устройства.

Happ не применяет облачные сервисы для размещения пользовательских настроек, не выполняет автоматическую синхронизацию между устройствами и не отправляет информацию о конфигурациях сторонним платформам.

Пользователь полностью контролирует работу с профилями: самостоятельно создаёт, импортирует, изменяет, сохраняет, удаляет и активирует необходимые конфигурации через интерфейс приложения.

Импортированные файлы используются исключительно внутри Happ для работы с настройками, не передаются другим пользователям и не обрабатываются внешними организациями.

# Особенности работы приложения

Happ выступает в качестве локального менеджера пользовательских конфигураций и не является поставщиком сетевых услуг.

Приложение не содержит:

* собственной серверной инфраструктуры;
* VPN-служб;
* прокси-систем;
* удалённых платформ обработки данных;
* встроенных каталогов готовых конфигураций;
* автоматических функций получения или распространения профилей.

Все необходимые параметры для работы с подключениями пользователь предоставляет самостоятельно — вручную либо посредством импорта заранее подготовленных конфигурационных файлов.

Happ не определяет назначение создаваемых профилей и не регулирует способы их использования. Пользователь самостоятельно отвечает за применение приложения в соответствии с законодательством и правилами, действующими в его регионе.
